## 排版
### 正常流排版
浏览器最基本的排版方案是**正常流排版**，包含了顺次排布和折行等规则。
#### 盒模型
content + padding + border + margin
#### 绝对定位元素
- 绝对定位元素把自身从正常流中抽出，直接由top、left等属性确定自身等位置，不参加排版计算，也不影响其它元素
- 绝对定位元素由position属性控制
#### 浮动元素
- 浮动元素是使自己中正常流的位置向左或者向右移动到边界，并且占据一块排版空间
- 浮动元素由float属性控制
#### 块级元素
总是单独占据一整行，一般为结构元素
